# NTUEE Communication Laboratory (Fall 2025)
This repository contains the coursework, laboratory implementations, and research paper reviews for the Communication Electronics Laboratory course at National Taiwan University (NTU), Department of Electrical Engineering, Fall 2025.

The course covers the practical implementation of digital communication systems, ranging from baseband signal processing to real-world transmission using Software Defined Radio (USRP) platforms.

## 2. Laboratory Overview

### Lab 1: Modulation and Demodulation

-  IQ modulation from baseband to radio frequency.

-  Optimal demodulation process.

### Lab 2: Up/Down-sampling and Filtering

-  Perform up-sampling and down-sampling.

-  Apply the Square Root Raised Cosine (RRC) filter.

### Lab 3: OFDM, Frame Synchronization & Equalization

-  Implement OFDM waveform generation and processing.

-  Implement Wi-Fi Short Training Sequence (STS) and Long Training Sequence (LTS) for frame
synchronization and equalization.

### Lab 4: OFDM Transmission Using USRP

-  Implementation of Wi-Fi-based OFDM transmission on USRP.

-  CFO estimation and correction.

### Lab 5: Cross-Device OFDM Transmission

-  Cross-USRP transmission and reception

-  Performance enhancement using multiple antennas.

## 3. Paper Reviews
Critical analysis of four state-of-the-art communication papers focusing on:

-  Summary: Core contributions and objectives.

-  Figure Analysis: Technical breakdown of simulation and experimental results.

-  Critical Review: Identification of strengths and potential weaknesses.

-  Debate Materials: Specialized presentation slides for the fourth paper review session.

## 4. Final Project: Distance Estimation via Yagi-Uda & Multipath

This project explores the spatial characteristics of the Yagi-Uda antenna and the physical properties of multipath propagation to conduct distance estimation. Including: 

-  Code: MATLAB implementation.

-  Report: experimental results & analysis.
